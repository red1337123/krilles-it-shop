# Projektplan – Krilles IT Shop

## Tidsplan

Projektet körs från början av maj till 17 maj då det ska redovisas, och publiceras 20 maj.

| När | Vad | Milstolpe |
|-----|-----|-----------|
| 2–4 maj | Projektplan, filstruktur, HTML-skelett och navigation | |
| 5 maj | Visa grundstrukturen för Christer nav, startsida, layout | Milstolpe 1 – Visa upplägg för Christer |
| 6–9 maj | Bygga klart undersidorna (tjänster, lager, TV & ljud, boka), innehåll och styling | |
| 10 maj | Visa alla sidor för Christer, kolla att innehållet stämmer | Milstolpe 2 – Genomgång av innehåll med Christer |
| 11–15 maj | Design, responsivitet, bilder, testa i webbläsare, tillgänglighet | |
| 16 maj | Sista koll med Christer innan redovisning | Milstolpe 3 – Slutgiltig genomgång med Christer |
| 17 maj | Redovisning | |
| 20 maj | Publicering | |

## User cases

1. Kunden vill se öppettiderna så hen vet när butiken har öppet.
2. Kunden vill kolla vilka datorkomponenter som finns i lager.
3. Kunden vill läsa om vilka reparationer och felsökningar som erbjuds.
4. Kunden vill veta att man kan lämna in sin dator för optimering.
5. Kunden vill boka tid för hemservice.
6. Kunden vill se vilka TV-apparater och ljudanläggningar som säljs.
7. Kunden vill hitta kontaktinfo och veta var butiken ligger.

## Designval

Sidan ska vara ren och modern, inspirerad av Apple stilen luftig layout, mycket whitespace, enkla färger och tydlig typografi. Ingen kopia av Apple, men samma känsla av att det är stilrent och inte för mycket.
